# 兒童權利公約 (CRC) 簡介與素養實踐 - 50頁互動簡報網頁

本專案為 **CRC (Convention on the Rights of the Child, 兒童權利公約)** 的 HTML5 互動簡報與素養試題網頁應用程式。

## 🌟 專案亮點與特色

1. **完整 50 頁簡報架構**：
   - **PART 1: 概述與核心精神 (1-7頁)**：CRC 起源、台灣《CRC施行法》國內法化、四大基本原則心智圖。
   - **PART 2: 四大基本原則深度解析 (8-15頁)**：禁止歧視、兒童最佳利益、生命生存與發展權、尊重兒童意見 (表意權)。
   - **PART 3: 4Ps 權利分類與主題 (16-27頁)**：生存權 (Survival)、發展權 (Development)、受保護權 (Protection)、參與權 (Participation)。
   - **PART 4: 迷思破除與國際趨勢 (28-34頁)**：三個任擇議定書、正向管教 vs 體罰、權利與義務迷思、數位時代 CRC (第 25 號一般性意見)。
   - **PART 5: 素養導向情境題 (35-45頁)**：**10 題素養導向情境選擇題**，涵蓋校園髮禁服儀、數位隱私、公園遊具、特教融合、氣候罷課、兒少新聞去識別化、親權爭奪、補習過度、午餐菜色與社群言論。附帶即時答案反饋與法理詳細解析。
   - **PART 6: 實踐、資源與結語 (46-50頁)**：校園與家庭行動清單 (Checklist)、113/1925/家扶等求助資源、權威文獻、總結省思與 Q&A。

2. **現代化 UI/UX 互動介面**：
   - **暗黑/明亮主題切換** (Dark/Light Theme Toggle)
   - **全螢幕簡報模式** (Fullscreen Mode)
   - **鍵盤快捷鍵控制** (方向鍵左右、PageUp/PageDown、空白鍵、Home/End)
   - **50頁縮圖網格總覽** (Overview Grid Modal)
   - **目錄搜尋與快速 jump 跳頁**
   - **即時測驗互動與詳解呈現**

## 🚀 快速開始 (Quick Start)

直接在瀏覽器開啟 `index.html` 即可使用，或使用伺服器（如 Live Server, VS Code）開啟。
